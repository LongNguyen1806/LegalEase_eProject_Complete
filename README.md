**Project Introduction**

LegalEase focuses on creating a transparent and automated legal environment:



Transparency: Verified profiles, authentic reviews, and clear service pricing.



Automation: Systematized availability and automated notifications for both clients and lawyers.



Practical Experience: Applying academic knowledge to professional real-world scenarios.



**Applied Technologies**

The system is built using a modern technology stack:



Backend: Laravel Framework.



Frontend: React (Vite).



Database: SQL Server / XAMPP.



Tools \& Infrastructure: Docker, GitHub.



**Summary of Core User Flows**

The platform revolves around four primary roles:



Guest: Quick search with versatile filters (name, location, expertise), viewing detailed lawyer profiles, and account registration.



Customer: Real-time booking by day/hour, integrated payment systems, appointment tracking, and service reviews.



Lawyer: Financial dashboard for cash flow monitoring, flexible schedule management, Q\&A interaction, and profile verification management.



Admin: User and lawyer verification, financial oversight (revenue and refunds), CMS management, and system security auditing.
